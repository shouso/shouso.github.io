[Tutz] Download dr RS/situs berlogin lainnya dg wget@Linux (Bonus script downloader)


Halo bro and sis, mo berbagi sedikit tulisan

"Tutz bagaimana men-download file dari situs yg mengharuskan kita login terlebih dahulu dengan wget@linux"

Contoh situs : RS, oracle dll

Kayanya sih udah pernah dibahas sama kaskuser, tp gpp lah, buat pelengkap aja 

Untuk mendownload dari situs yg perlu login, sebagian besar situs hanya mengecek cookie dari browser yg kita pake apakah sudah ada ato belon

Klo sudah ada, cek cookie untuk diverifikasi ke server, klo belon, tampilkan login screen

So, the trick is :

1. Kita harus bikin cookie (klo belon ada)
2. Pake cookie yg di no. 1 setiap kita buka situsnya

Hayu kita mulai, kita ambil contoh untuk situs RS

Perlengkapan
1. lynx text based browser
2. wget
3. link yg mo di download

Langkah²
1. Ketikkan perintah ini di shell
Code:

cat > lynx.cfg

Kemudian isi dengan
Code:

SET_COOKIES:TRUE

PERSISTENT_COOKIES:TRUE

ACCEPT_ALL_COOKIES:TRUE

COOKIE_FILE:~/.lynx_cookies

INCLUDE:/etc/lynx.cfg

Kemudian teken
Code:

Ctrl + d

note : lokasi file cookie-nya terserah mo ditaro dimana, disini ane naronya di home dir user yg lg dipake

2. Buka link yg mo di-download menggunakan lynx
Misal kita mo download
Code:

http://rapidshare.com/files/109426323/ChildrenOfHeaven.part1.rar

Kita ketikkan ini di shell
Code:

lynx -cfg=./lynx.cfg http://rapidshare.com/files/109426323/ChildrenOfHeaven.part1.rar

Klo udah, tar bakal muncul kurang lebih kaya gini
Spoiler for Tampilan RS:




Kursor / pointer browsernya yg warna merah (liat gambar)

Kita arahin tu kursor ke "Premium user" pake tombol panah, teken "enter"

Klo udah, bakal muncul halaman login kurang lebih kaya gini
Tampilan login RS:



Arahin kursor pake tombol panah ke form username, isi username ente
Trus pijit tombol panah bawah, masukin password ente
Trus pijit tombol panah bawah lg, teken "enter" buat login

Klo udah, tar muncul tampilan kurang lebih kaya gini
Tampilan setelah login RS:



Kita ga usah download filenya, krn tujuan kita pan buat bikin cookie aja 

Pijit tombol "q" buat keluar dari browsernya, pilih "y" klo ditanya macem²

Sekarang kita cek file .lynx_cookies nya, ketikkan ini di shell
Code:

more .lynx_cookies

Tar isi dari file .lynx_cookies kurang lebih kay gini
Code:

.rapid*share.com TRUE / FALSE 1731510000 enc <kode² gajebo>

note: <kode² gajebo> punya ane, ane umpetin 

Sekarang, untuk download pake wget-nya, tinggal ketikkan perintah ini di shell
Code:

wget --load-cookies=.lynx_cookies http://rapidshare.com/files/109426323/ChildrenOfHeaven.part1.rar

Klo mo wget-nya work in the background, tambahin -b, jadi kaya gini
Code:

wget -b --load-cookies=.lynx_cookies http://rapidshare.com/files/109426323/ChildrenOfHeaven.part1.rar

Done, kita tinggal nunggu downloadan selesai 

Kita juga bs nambah cookie situs yg laen ke file .lynx_cookies, tinggal ulangin aja langkah ke 2

Yup, tinggal buka situs yg laen aja. Insya Allah cookienya bertambah secara otomatis ke file .lynx_cookies (asal ada opsi -cfg=./lynx.cfg)

BONUS !!!!

Script mass download and work in the background buat linux
Code:

for link in $(cat $1)

do

wget --load-cookies=.lynx_cookies -b $link

done

Cara make
1. Buat file yg isinya daftar downloadan kita, misal nama filenya "daftar.txt"
2. Buat file yg isinya script bonus di atas, misal namanya "download"
3. Ubah permission file "download" supaya bs di-eksekusi : "chmod +x download"
4. Untuk mulai download, ketikkan : "./download daftar.txt"
