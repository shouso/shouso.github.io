Baik yang berbayar serta mahal maupun yang berlisensi Free Software.Berikut ini adalah aplikasi freeware yang wajib dicoba bagi para pemakai PC berbasis windows yang membutuhkan aplikasi yang andal tetapi memiliki dana terbatas (seperti saya) :

1. OpenOffice 3.2, wajib dicoba, lebih baik dibandingkan versi sebelumnya (3.0) dan tidak kalah dengan versi terbaik MS Office (Office 2003), dengan tampilan yang tidak kalah dengan perangkat kantor berbayar. kelemahan utama adalah waktu loading pertama yang sedikit lama.
2. GIMP (GNU Image Manipulation Project), pengganti terbaik untuk pengolah gambar berbasis piksel semacam Adobe Photoshop yang mahal.
3. Blender, aplikasi untuk penghobi animasi dan gambar 3 dimensi.
4. WAMP (Windows, Apache, MySQL, PHP), server suite bagi pengembang aplikasi berbasis web, maupun yang ingin mencoba hebatnya aplikasi database MySQL. Dilengkapi dengan phpMyAdmin dan SQLite Manager yang digunakan untuk manipulasi MySQL berbasis GUI, sehingga memudahkan membuat database.
5. Flightgear, games simulasi pesawat terbang yang sangat indah, patut dicoba bagi penggila games strategi.
6. Clamwin, antivirus berbasis ClamAV (antivirus untuk linux), yang cukup andal dengan update database virus yang mudah.
7. Netbeans, IDE berbasis Java Development Kit (JDK) yang dapat digunakan untuk membangun aplikasi dengan berbagai bahasa program (PHP, JAVA, Ruby, C, dll). Membutuhkan JDK serta berukuran cukup besar (200 MB).
8. Audacity, audio editor yang dapat digunakan untuk mengedit dan memanipulasi file audio serta dapat menangani file audio berjenis mp3, ogg, dan yang lain. file hasil editan dapat langsung diekspor menjadi file berekstensi mp3, ogg, maupun disimpan dalam file audacity (au).
9. Lupo Pensuite, suite aplikasi portabel yang berisi hampir seratusan aplikasi baik yang freeware maupun yang hanya shareware, mulai dari multimedia, office, security, maupun games.

Sementara, itulah daftar Freesoftware saya, yang tidak kalah bagus dibandingkan aplikasi berbayar sejenis. selamat mencoba!!!
