hobi donlot dari youtube, dan pake Linux?pastinya tau dong youtube-dl?apalagi yang make ubuntu, dah ada tuh paket yang masuk di reponya ubuntu. nah yang make debian mesti bersabar nih, karena masih masuk di repo testing, belum masuk ke stable. tapi kalo mo nyoba sih oke-oke aja. tinggal donlot dair repo testing, terus install pake dpkg -i. beres deh? kalo mau ngebangun dari source code, tinggal donlot dari github. lalu kompilasi manual.

eh, sekarang lebih gampang lagi ternyata, meskipun belum masuk repo stable nya Wheezy, tetep bisa di pake kok, caranya donlot, dan  bikin jadi dapat dijalanin sebagai aplikasi dengan ketik ini

wget http://youtube-dl.org/downloads/2013.08.14/youtube-dl -O /usr/local/bin/youtube-dl
chmod a+x /usr/local/bin/youtube-dl

kalo biasa make pip, bisa juga install langsung dari reponya python, karena youtube-dl ini cuma skrip python dengan mengetik  pip install --upgrade youtube_dl

Cara makenya mudah banget lho? tinggal ketik di terminal youtube-dl nama url di youtubenya, misal mo donlot lagunya Maher Zein yang Insha Allah tinggal ketik youtube-dl http://www.youtube.com/watch?v=KfXIF2Mm2Kc. lalu tunggu selesai...
