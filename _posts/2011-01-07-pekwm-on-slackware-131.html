setelah melihat deski minimalis om tuxo jadi kepengen juga ngerasain pekwm di slackware 13.1. jadilah nyari lagi di slackbuild.org ada ga skrip instalasinya. dan memang ada sih... malah kalo di slackware mau instalasi pekwm tinggal install aja, ga butuh dependensi lain karena emang udah ada..beda kalo mo nambah window manager lain kayak wmii yang harus install beberapa paket lain.


katanya gini di slackbuild.org tentang pekwm


"This window manager doesn't require any packages that aren't
already provided by Slackware. Just run the SlackBuild and have fun :) :D"


maka jadilah ku install pekwm dengan mantra ajaib lagi setelah selesai donlot source tentunya...


Nah, selesai install, ternyata aku belum bisa menikmati desktop  minimalis pekwm karena emang harus di buatin berkas (file) konfigurasi sendiri pekwm nya. pekerjaan ini harus dilakukan secara manual. tetapi banyak panduannya kok... tenang saja. bisa dari dokumentasi resminya pekwm, atau bisa juga dari sumber yang  lain.  Nah apa saja yang harus disiapkan buat menikmati pekwm, inilah langkah-langkahnya :


1. install dulu pekwmnya (tergantung distro anda pastinya)


kalo di slackware ya installpkg /tmp/pekwm* (misalnya abis build pake skrip slackbuild).


kalo arch gini



pacman -S pekwm

2. buat skrip untuk masuk ke pekwm, kalo udah ada kdm/gdm tinggal tambahin satu file di folder /usr/share/xsessions dan kasih nama Pekwm.desktop. Isinya sbb



[Desktop Entry]
Encoding=UTF-8
Name=PekWM
Comment=Start PekWM
Exec=/usr/bin/pekwm
Icon=
Type=Application

atau bisa tambahin



exec pekwm

di file /.xinitrc ente.


3. buat folder .pekwm di home sodara-sodara untuk menampung konfigurasi pekwm, isinya yang penting adalah sbb :
1. folder themes buat menampung tema-tema pekwm

3. file menu, yang berisi menu-menu yang pengen ente tampilin pas make pekwm, contohnya



# Menu config for pekwm

# Variables
$TERM = "xterm -fn fixed +sb -bg black -fg white"
