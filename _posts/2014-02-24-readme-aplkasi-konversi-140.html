APLIKASI KONVERSI 1.0.3.2 build 11-02-2014
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

DESCRIPTION:

Aplikasi untuk konversi ADK SPM dan Kontrak dan dipasang di Kantor Pelayanan Perbendaharaan Negara yang sudah menggunakan SPAN. 

SYSTEM REQUIREMENTS:

- Windows (XP minimum)

CHANGES:

- 1.0.3.2 build 11-02-2014
  - Penambahan menu Referensi Bank Pusat;
  - Penyesuaian mapping NIP penerima untuk supplier tipe 6.
- 1.0.3.2 build 07-02-2014
  - Perbaikan konversi ADK SPM terkait SPM kontraktual dan non kontraktual dalam satu ADK.
- 1.0.3.2 build 06-02-2014
  - Penyesuaian nama penerima di ADK PMRT dan nama supplier di ADK supplier terkait panjang/jumlah karakter (khusus tipe supplier 1);
  - Perbaikan pembentukan ADK suplier tipe 1 terkait satker perwakilan luar negeri.
- 1.0.3.2 build 05-02-2014
  - Perbaikan ADK suplier tipe 3 dan PMRT dari ADK SPM non lampiran terkait kode NIP (req ADK dari Aplikasi SPM 14.1.2);
  - Perbaikan ADK suplier tipe 3 dan PMRT dari ADK SPM non lampiran terkait NPWP Bendahara Pengeluaran.
- 1.0.3.2 build 04-02-2014
  - Perbaikan ADK suplier tipe 3 dari ADK SPM penerima terlampir terkait kode negara asal bank;
- 1.0.3.2 build 02-02-2014
  - Perubahan mapping COA SEGMENT5 terkait SPMIB;
  - Perubahan mapping kode jenis dokumen terkait SPM uang muka kontrak, semula 86 menjadi 81;
- 1.0.3.2 build 24-01-2014
  - Penambahan mapping jenis dokumen untuk jenis SPM 05 kode sifat SPM 6;
  - Perbaikan ADK release kontrak multiyear.
- 1.0.3.2 build 24-01-2014
  - Perubahan COA SEGMENT10 terkait SPMKP untuk pengeluaran, semula 2 (angka dua) menjadi 7 (angka tujuh);
  - Perbaikan scan barcode SPM.
- 1.0.3.2 build 23-01-2014
  - Perubahan COA SEGMENT10 terkait SPMKP untuk pengeluaran, semula 2 (angka dua) menjadi 7 (angka tujuh).
- 1.0.3.2 build 17-01-2014
  - Perbaikan konversi ADK kontrak terkait kode dan nama cabang bank.
- 1.0.3.2 build 16-01-2014
  - Perbaikan konversi ADK kontrak terkait kode dan nama bank luar negeri.
- 1.0.3.2 build 15-01-2014
  - Perbaikan konversi ADK kontrak terkait valas.
- 1.0.3.2 build 07-01-2014
  - Penambahan menu update referensi;
  - Perbaikan ADK Supplier tipe 3.
- 1.0.3.2 build 02-01-2014
  - Release 2014.

INSTRUCTIONS:

- Pemasangan database
  - Stop service SP2D;
  - Jalankan file setup_database_konversi.exe di server SP2D.
- Pemasangan aplikasi
  - Jalankan file setup_aplikasi_konversi.exe di semua PC pengguna aplikasi.
- Menggunakan aplikasi
  - Silahkan baca file help.

FAQ:

Segala pertanyaan terkait Aplikasi Konversi dapat menghubungi service desk SPAN.

CREDITS:

Aplikasi Konversi adalah produk dari Direktorat Sistem Perbendaharaan dan Direktorat Transformasi Perbendaharaan.

THANKS TO:

Semua pihak yang terlibat dalam pengembangan Aplikasi Konversi.
