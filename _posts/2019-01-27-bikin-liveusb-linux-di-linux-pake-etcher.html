Dulu kalo yang suka mencoba-coba distro linux itu pasti boros CD/DVD karena dulu distro linux yang support LiveUSB itu masih bisa diitung dengan jari, apalagi diawal tahun 2004 ketika saya pertama kali kenal linux. Install linux itu identik dengan layar item pake terminal alias command promt. Tapi waktu itu udah ada Knoppix yang liveCD.

Teknologi berkembang, sekarang ini sudah gampang sekali mencoba distro linux via LiveUSB sebelum melakukan instalasi ke PC/Laptop. Alat untuk membuat LiveUSB juga makin berkembang, kalo dulu membuat liveUSB di linux itu identik dengan command d, sekarang udah ada GUI nya namanya Etcher buatan resin.io, ini juga aplikasinya portabel menggunakan Appimage. silahkan kunjungi etcher.io untuk lebih mengenalnya dengan intim... dddddd.

Karena etcher pake appimage, jadi tinggal donlot di webnya etcher, terus ekstraksi dan beri kewenangan agar bisa dijalankan sebagai program. Lalu jalankan aja, kalo pertama kali biasanya ditanyaain mau di install apa kagak. Kalo di install, nanti ikon etcher akan muncul di daftar software seperti ini :


 Etcher di Gnome Shell

Terus make nya gimana? lebih gampang lagi, asal udah disiapkan file .ISO distro linuxnya, tinggal jalankan, pilih, USB Drive nya, terus klik flash. Nanti diminta masukin password, terus tunggu sampai flashing nya selesai.

Kal udah tinggal dicoba aja liveUSB Linuxnya.

Ohya, peringatan ini etcher sepertinya pake dd sebagai backbonenya, jadi setelah di flash, semua data di usb ente akan ilang ketimpa semua. jadi pastikan tidak ada data penting sebelum UFD Flashdisk nya mau digunakan untuk liveUSB pake etcher.
