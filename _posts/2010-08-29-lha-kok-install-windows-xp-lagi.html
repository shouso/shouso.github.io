Senin,  30 Agustus 2010


itulah yang terjadi...


setelah beberapa lama ga pernah install Windows XP, tiba - tiba hari ini daku harus melakukan instalasi Windows XP lagi, ke Netbook HP Mini yang hanya ada Freedos dan tidak ada CD Drive nya lagi....


Kalo punyaku sendiri sih tak isi sama Ubuntu Net Remix yang katanya yahuud deh...tetapi apatah daya ini Netbook punya pak Parji, teman kantor yang maunya di install XP demi kemudahan dalam bekerja...


Kenapa.... Karena hampir seluruh aplikasi buatan Direktorat Sistem Perbendaharaan, Ditjen Perbendaharaan dikembangankan under Windows.... ga ada yang dikembangkan untuk non WIndows, kode sumbernya pun yang tahu hanya para pengembang di Dit. SP sono... jadilah pak Parji harus mau tak mau pake MS Windows XP....


Nah.... karena ga ada CD Drivenya dan Pak Parji and the gang di kantor ga ada yang punya CD Drive eksternal, maka untuk test drive daku boot dahulu lewat Slax 6, hasilnya ga mengecewakan, KDE 3.5 muncul dengan baik dan dapat digunakan.


Nah... dengan analogi instalasi via USB yang sering daku lakukan untuk Linux, maka mulailah pencarian bagaimana sebaiknya proses instalasi Windows XP via USB dan bagaimana membuat USB Drive Win XP bootable.  Mulailah aku bertanya kepada simbah google tentang bagaimana instalasi nya dan hasilnya luar biasa.... ada 8,9 Juta list ketika ku ketikkan "installing windows xp from usb drive" di google... benar-benar hasil yang luar biasa...


ada tiga software yang harus disiapkan untuk membuat USB drive bootable Windows XP, selain CD Windows XP tentunya, yakni :


1. usb-prep8, digunakan untuk menyiapkan USB drive, formatting dll


2. petoUSB,


3. dan bootsect digunakan sebagai sarana mengubah USB agar dapat dibooting.


selain terpisah-pisah ada juga yang disatukan dalam satu folder bernama winxp_usb yang isinya juga ketiga software diatas.


prosesnya dapat dijelaskan sebagai berikut :


1. pertama persiapan USB drive, dengan usb_prep8,


2. membentuk tempimage dan Temporary file di T:\ sebagai tampungan data dari CD Windows.


3. selanjutnya peto USB melakukan copy ke USB...


cukup lama prosesnya... dan membosankan....


setelah berhasil membuat USB bootablenya, baru masukkan USB drive ke soket USBnya dan silakan hidupkan....tekan ESC... dan pilih boot from USB atau pencet f9... tapi kok susah ya....setelah itu anda harus pilih mode text dulu, baru setelah dicopi ke harddisk, reboot masuk ke mode GUI...itupun belum tentu berhasil.


Setelah capek nyoba instalasi lewat USB dan ga berhasil, tiba pilihan terakhir , pak Parji pinjem cd drive eksternal dari teman dan cd Windowsnya sekalian, barulah sukses di install.....


Tapi ya kok ribetnya belum selesai, setelah sukses intalasi Win XP nya, driver vga dan macem-macemnya kan belum ada? lha gimana ini, kan cuma ada freedos doang, terpaksalah pak parji harus bertanya lagi ke simbah untuk melengkapi apa yang kurang dari Win XP nya..... duh kok ya repot amat....
