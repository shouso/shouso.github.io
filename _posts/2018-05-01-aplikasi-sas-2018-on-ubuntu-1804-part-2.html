Ngelanjutin postingan kemarin, berikut ini langkah-langkah instalasi Aplikasi SAS 2018 di ubuntu 18.04, sbb :

Prasyarat (ini spek laptop saya ding)
hardware : ASUS  X45U, AMD Brazos APU, 8 GB RAM, 500 GB HDD.
Software  :  Ubuntu Bionic Beaver 64 bit, wine 3.0 dan winetricks  terbaru.
Internet, boleh yang 4G ber GB itu.
Langkah pertama adalah instal dulu Ubuntu 18.04 nya... hehehe... becanda...

Langsung aja;
1. aktifkan repository universe multiverse restricted plus canonical partner nya via software dan update. (klo biasa edit text cari sendiri ya), terus update reponya..
2. install wine dan winetricks pake Ubuntu Software boleh atau ketik di terminal juga boleh.
apt install wine winetricks
Tunggu sampe selesai..
3. install librarinya windows pake winetricks

winetricks corefonts eufonts tahoma cjkfonts 
ikuti perintahnya

winetricks vb6run 
ikuti perintahnya juga
  
winetricks mdac28

winetricks msxml4 mfc42 jet40 native_oleaut32

4. Install Database Aplikasi SAS 2018 sama Aplikasi SAS 2018 nya.

5. install mysql-odbc-connector bawaannya Aplikasi SAS 2018, caranya pindah direktori ke foldernya aplikasi SAS 2018

   cd /home/usernameanda/.wine/drive_c/AplikasiSAS2018

terus instal pake winetricks

winetricks wine msiexec /i mysql-connector-odbc-5.1.8-win32.msi

ikuti perintahnya sampe selesai.

Baru coba buka aplikasi SAS 2018 nya... akan ada beberapa warning, LOITEMS gitu dll, abaikan aja. kalo sukses maka akan masuk ke aplikasi SAS 2018 nya.

Jika sukses baru install update nya Aplikasi SAS 2018.

Semisal ga mau masuk lagi dengan warning "TIDAK DAPAT TERKONEKSI DENGAN DATABASE", matiin service mysqld.exe*32 pake task managernya WINE dengan ngetik wine taskmgr. terus coba lagi lakukan update atau masuk ke aplikasi SAS nya.
