Sawfish apakah itu?sawfish adalah nama jenis ikan dari keluarga hiu-hiuan orde yang hidungnya berbentuk gergaji. Sawfish adalah jenis ikan pari dengan ornamen gergaji, tetapi jangan keliru sama sawsharks dari jenis hiu ya?Gergajinya itu digunakan sebagai alat sensor untuk mencari makan. Inilah salah satu jenis dari sawfish: nah kalo yang ini adalah gambar sawsharks


Eh, tapi kita bukan mau ngebahas taksonomi sawfish ding, tetapi mau membahas salah satu window manager di GNU/Linux yang jarang kedengeran kiprahnya. dan kebetulan namanya Sawfish, logonya pun gambarnya si ikan gergaji, seperti berikut ini :


Sawfish yang ini adalah salah satu Window Manager yang pernah dipake sebagai WM nya GNOME. Sampai sekarang pun sawfish tetap ada di reponya distro/distro besar yang memakai GNOME sebagai DE utamanya seperti Debian dan Ubuntu.


Untuk distro berbasis debian, instalasinya pun gampang, yakni tinggal etik perintah ajaib



apt-get install sawfish

tinggal tunggu selesai dan konfigurasinya.


Lain halnya dengan Slackware yang memang tidak menyediakan paket repository diluar paket yang ada di installernya. Maka kalo pengen memakainya ya harus install lewat source codenya, atau nyari di reponya slacky.ue. tapi adanya untuk slack 12.1. Paling aman sebenarnya compile sendiri dengan bantuan skrip slackbuild.org


Sayangnya sawfish tidak ada skrip slacbuilldnya, makanya satu-satunya cara ya instalasi manual.


Untuk menikmati, sawfish ada 4 paket yang harus di install yakni librep, rep-gtk, sawfish, dan swafish-pager.


Cara instalasinya pun sama dengan paket source code yang lain yakni dengan mantra berikut



./configure
make

make install

Untuk instalasi rep-gtk memerlukan tambahan mantra berikut ini kata wikianya sawfish:
/configure \
  --prefix=/usr \
  --without-gnome \
  --without-libglade \
  --sysconfdir=/etc \
  --localstatedir=/var \
  --with-perl=/usr/bin/perl \
  --build=$ARCH-slackware-linux \
  --host=$ARCH-slackware-linux

dan ini pas mau kompilasi
make "rep_LIBTOOL = /usr/libexec/rep/i486-slackware-linux-gnu/libtool --tag=CC"

pas pertama kali yang d install adalah librep, karena dibutuhkan oleh repgtk.


nah masalah mulai muncul setelah sukses instalasi rep-gtk, secara default hasil kompilasi dan install dari librep di taro di folder /usr/local. pas configurasi rep-gtnya ga bisa menemukan dimana si librepnya. dan saya ga nambahin opsi seperti yang diatas, makanya berkali-kali gagal pas instalasi rep-gtk.


Hampir putus asa, lalu mencari cara lain yakni menggunakan skrip slackbuild yang ada di slacky.ue untu kompilasi sawfish di Slackware 12.1


gagal juga, karena emang aga beda dengan skrip yang untuk 13.1
