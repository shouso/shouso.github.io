Nasib... nasib...jadi poor man memang ga enak banget...mau install Debian Squeeze aja jadi kesusahan.

Definisi poorman disini adalah PCnya ga bisa bootin lewat USB port, dan ga ada Floppy sama ga ada CD/DVD ROM.

Jalan satu-satunya ya harus install lewat partisi Harddisk dong.. dan untungnya Debian menyediakan pilihan itu.

Lalu gimana sih langkah-langkah poorman install Squeeze, em.. minimal kayak gini deh...:

1. Prasyarat poorman install

- udah ada OS nya sebelumnya dan berfungsi

- ada satu partisi khusus buat menmpung file ISO dan bootnya.

- download vmlinuz dan initrd.gz khusus buat hd install dari sini hd-media

2. format dulu partisinya, janga lupa di buat bootable terus copi file iso dan vmlinuz plus initrd.gz ke partisi tadi, jangan lupa buatin folder boot buat nampung tiga file tadi.

oya lupa, katanya wiki.debian LiLo ga mau booting dari NTFS, jadi jangan dformat NTFS ya?

3. edit file /etc/lilo.conf atau /boot/grub.cfg buat nambahin partisi tadi.

kalo udah coba di update lilo sama grubnya.

kalo lilo cukup ketik lilo, kalo grub ga usah diapa-apain.

3. udah itu silakan reboot... nah muncul ga di layarnya lilo sama grub?pilih pilihan booting yang barusan kita buat dan silakan lanjutkan instalasi seperti biasanya...

semoga berhasil...
