Yang hobi bikin live cd or live usb linux pasti ga asing lagi sama tool tool yang dibuat untuk mempermudah proses membikin liveusb; jaman gini masih make CD..:D. Jumlahnya juga ga sedikit, banyak malah, dari yang cuma jalan di make command line macem dd sampe yang GUI nya yahud dan bisa sekalian nyariin virtualmachine kayak si LiLi alias LinuxLive.
Nah, Postingan kali ini membahas apasih alat-alat yang bisa digunakan untuk membuat liveusb linux. Berikut ini daftarnya :
1. dd 
ini command line yang umum di Unix dan Linux, fungsi utamanya untuk convert dan copy file. Tergantung opsi yang dipilih saja. salah satunya bisa digunakan untuk menulis berkas ISO linux ke USB Flash Disk pake parameter "if=" "of=". cuma emang harus hari hati, UFD target mesti kosong karena perintah ini menghapus semuanya di target. contoh penggunaan :
dd if=usbboot.img of=/dev/sdx bs=512
 
 
# dari README.TXT nya slackware...
 
penggunaan umumnya seperti ini :
 
dd [bs=SIZE[SUFFIX]] [count=BLOCKS[SUFFIX]] if=FILE of=FILE [seek=BLOCKS[SUFFIX]]
 [skip=BLOCKS[SUFFIX]] [--size] [--list] 
[--progress] 
klo mau install Slackware pake FD mesti make perintah ini nih paling gampang.

2. Linux Live USB Creator
Ini salah satu USB Creator jempolan, gampang, mudah, dan GUI nya unix, transparan
gitu. Terus lumayan banyak fiturnya, salah satunya bisa sekalian nyariin mesin 
virtualbox untuk ngejalaninnya di Windows. Kalo distronya di dukung oleh skripnya,
bisa juga dibikin persistent. ada juga versi portabelnya.
 
Cuma sayangnya tidak multi platform alias khusus MS Windows.
 
3. Unetbootin
 
Kelebihannya multiplatform, bisa dipake di MS Windows, beberapa distro Linux 
(dari Fedora sampe Gentoo), termasuk MacOS.Ga perlu install, karena memang portabel,
dan kalo distro tertentu yang didukung bisa sekalian ngedonlotin
ISO nya.. kalo ada internet..:D
Kekurangannya, GUI nya biasa aja...
 
Next Read :
http://unetbootin.sourceforge.net/
 
4. Pendrivelinux.com
ini website yang berisi kumpulan beberapa tool yang bisa untuk bikin USB Installer
linux,paling terkenal Universal USB Installer sama YUMI. Termasuk juga tutorial 
bikin USB installer make tool itu.
