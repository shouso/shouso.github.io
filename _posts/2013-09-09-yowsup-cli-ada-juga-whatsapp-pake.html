Katanya whatsapp make protokol xmpp yang dimodif ya, jadi meskipun terkesan tidak ngebuka kode sumbernya, masih ada juga yang berhasil membuat clientnya dengan memanfaatkan API yang saya sendiri juga ga ngerti dapatnya dari mana. Pengembang itu adalah Tarek Galal yang memanfaatkan Python dan membuat yowsup yang berfungsi sebagai jembalan untuk berhubungan dengan server whatsapp. Nah, untuk dapat berkirim pesan via whatsapp di Linux memakai yowsup, paling gampang pake antar muka yang disediakan oleh yowsup yakni yowsup-cli.Emang sih, udah ada yang bikin plugin whatsapp untuk pidgin tapi kok lebih gampang nyobain pake yang cli ya, sekalian belajar ekren ala hacker gitu... hihihi... becanda...Lha caranya gimana? gampang install aja yowsup-cli dan paket yang dibutuhkan sebagai berikut ini :

1. install dulu dependensi yowsup nya
apt-get install python python-dateutil python-argparse git


2. donlot yowsup nya pake git atau pake wget
git clone http://github.com/tgalal/yowsup.git
wget https://github.com/tgalal/yowsup/archive/master.zip

3. klo pake wget ekstrak dulu terus pindah ke folder ./yowsup-master/src klo pake git langsung pindah aja ke folder ./yowsup/src
unzip master.zip
cd yowsup-master/src

4. registrasikan nomor hape ente di https://coderus.openrepos.net/whitesoft/whatsapp_sms agar dapat sms code dari whatsapp

5.udah gitu edit yowsup-cli.config nya terus di save
cp config.example yowsup-cli.config
scite yowsup-cli.config

cc=62
phone=6285xxxx (nomor ente)
id=
password=

6. kasih permission agar skrip yowsup-cli dapat dijalankan, dan jalanin yowsup-cli untuk register ke server whatsapp
chmod +x yowsup-cli
./yowsup-cli --register 123-345 --config yowsup-cli.config
ganti 123-345 dengan isi sms kode dari whatsapp tad, nanti akan ada balasan seperti contoh ini

status: ok
kind: free
pw: S1nBGCvZhb6TBQrbm2sQCfSLkXM=
price: 0,89
price_expiration: 1362803446
currency: EUR
cost: 0.89
expiration: 1391344106
login: 34123456789
type: new
 masukkan isi pw dari balasan whatsapp tadi ke yowsup-cli.config

7.nah, udah siap untuk berkirim dan menerima pesan via whatsapp, klo mo kirim tinggal ketik kek ini

./yowsup-cli --send 62856778823 "contoh pesan whatsapp via yowsup-cli" --wait --config yowsup-cli.config
6288xxx itu nomor yang mau kita kirimin pesan, yang tentunya punya aplikasi whatsapp, dan pesannya ditulis setelahnya di apit sama tanda dua petik diatas.

mo liat ada pesan tinggal ketik 
./yowsup-cli --listen --autoack --keepalive --config yowsup-cli.config

kalo mo chatting sama satu orang tinggal ketik 
./yowsup-cli --interactive 6288888877 --wait --autoack --keepalive --config yowsup-cli.config

628555xxx  itu nomor yang mau kita ajak chatting 

asyikkan... hehee.. cuma kelemahannya, ga bisa realtime terus, karena klo ga ada pesan biasanya disconnected, dan kalo pake cli ini saya belum nemu gimana kalo mau chatting dengan grub... ok segitu dulu...
