siapa bilang dekstop sederhana yang hanya memakai window manager ringan kayak PekWM ga bisa dipercantik?bisa dong, ga harus make gnome atau KDE atau enlightment buat tampil cantik. Pake PekWM pun desktop linux anda juga dapat tampil cantik kok. Dengan sedikit usaha lebih sih dibandingkan jika ingin memodifikasi tampilan GNOME maupun KDE.


Setelah selesai melakukan konfigurasi PekWM, dan bisa masuk ke pekwm, sekarang tinggal tugas anda adalah mempercantiknya sehingga tidak terkesan buram dengan warna hitam dan tanpa ada ornamen-ornamen.


Jika rumah dipercantik dengan ornamen-ornamen berupa lukisan, hiasan dinding, tanaman hias, dll, maka desktop linux apapun pilihan DE/WM nya juga punya ornamen-ornamen yang bisa digunakan agar desktop anda secantik rumah anda.


Unutk pekwm, ada beberapa aplikasi yang bisa menjadikan dekstop anda penuh ornamen:


1. nitrogen


karena pekwm hanya window manager, maka tidak dapat menampilkan wallpaper tanpa bantuan paket lain. nitrogen adalah salah satu aplikasi yang dapat digunakan untuk menampilkan gambar favorit anda sebagai kertas dinding dekstop linux anda. berikut ini adalah penampakannya..


Cara installnya sebagai berikut :


a. buat debian dan turunannya


pastikan repository anda benar, dan terhubung internet atau ada DVD reponya. Lalu ketikkan


 



sudo apt-get install nitrogen

tunggu sampai selesai prosesnya.


b. buat Arch linux tinggal ketik :


 



pacman -S nitrogen

c. buat slackware dan anak turunannya, cari dulu  skript SlackBuild nya di Slackbuild.org sekalian source code nitrogennya,lalu ekstrak skriptnya, dan pindahkan source code nitrogen ke folder hasil estrakkan tadi, lalu ketik:


 



cd ./nitrogen

./nitrogen.SlackBuild

tunggu sampai selesai dan terbentuk file nitrogen.xxx.tgz di /tmp.


lalu tinggal install dengan perintah :


 



installpkg /tmp/nitrogen*

2. tint2


nah ini dia salah satu paket paling bermanfaat untuk pekwm yang ga punya panel. installnya sama kayak nitrogen. terus cari skript konfigurasi tint2, untuk mempercantiknya, salah satu yang paling oke adalah tint2 brit. Ekstrak dan taruh tint2rcnya di folder /home/user/.config/tint2. bisa juga pake konfigurasi kreasi anda sendiri.
