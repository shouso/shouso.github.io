Salah satu kekurangan Gnu/Linux itu belum dilirik sama pengembang aplikasi pendukung pelaksanaan APBN agar dibikinin versi Linuxnya. Padahal Aplikasi SAS ( aplikasi Utamanya) itu make basis data MySQL yang notabenenya ada bau FOSS nya, tapi sih emang antarmuka nya masih make foxfro yang hanya ada di MS Windows.

Ini jadi masalah karena saya hanya make linux di Laptop saya, padahal sebagai lelaki panggilan itu saya kadang mesti ngerjain itu dirumah.

Apalagi pas cuti agak lama, ga enak juga kalo ninggalin semua tugas CSO di kantor, jadinya kemarin kepikiran bisa ga sih install Aplikasi SAS via Wine di Ubuntu 18.04? Kebetulan saya baru install lagi ubuntu setelah sekian lama pake Debian.

Permasalahan utama dari instalasi Aplikasi SAS di wine adalah mysql connector nya takut ga nyambung sehingga meskipun database dan aplikasi sas nya keinstall, tetap ga mau jalan.

iseng googling eh ketemu http://www.sanitarium.co.za/ yang ngasih trik agar mysql connectornya bisa jalan. Di dukung Ubuntu yang lebih bebas make non-free software ketimbang maknya Debian, makanya install Aplikasi SAS 2018 kemungkinan besar sukses... dan tada... emang sukses. Langkahnya secara garis besar tinggal install wine winetricks sama librari windows yang dibutuhkan oleh wine via winetricks, terus install aplikasi sas terakhir install mysql connector.

setelah dicoba keluar kek ini


artine sukses.... selamat selamat....
