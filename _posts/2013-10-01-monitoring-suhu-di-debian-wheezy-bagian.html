Distro Linux itu kaya akan alat-alat yang dapat digunakan untuk tujuan mempercantik tampilan desktop sekaligus memantau kondisi hardware, seperti temperatur, RAM, Harddisk, atau koneksi internet. Kalo di Windows kita sangat dimanjakan dengan Rainmeter dan Sammurize, maka ada Conky atau Screenlets di Linux. Belum lagi gkrellm, atau toy (program kecil) bawaan masing-masing Desktop Environment.

Nah, klo conky itu cenderung membutuhkan sedikit usaha untuk mendapatkan tampilan yang wah,sedangkan kita hanya butuh menampilkan hanya besaran suhu saja, ada baiknya mainan bawaan dari DE yang kita manfaatkan. Karena standarnya DE di Wheezy adalah Gnome 3 dengan Gnome Shell, maka sebaiknya gunakan saja Gnome Shell Extensionnya saja, agar hemat usaha dan tidak terlalu sudah melakukan konfigurasinya.

Nah, tentunya syarat utamanya adalah anda memakai Gnome Shell, untuk dapat memanfaatkan fitur dari gnome shell extensionnya. Setelah itu tinggal install ubo rampenya, donlot sensors extension, dan kompilasi lalu instal..terus tampilkan di gnome shell... gampang kan? hehehe....

lalu ubo rampenya apa aja? gini nih legkapnya :

1. install dan konfig lm-sensors
apt-get install lm-sensors

terus, lakukan konfigurasi dengan mengetik 

sensors-detect

jawab yes aja... sampai selesai.

udah itu ketik sensors klo udah dapet balasan seperti dibawah, berarti sukses

klo langkah satu udah, sebenarnya kita udah dapat mengecek suhu penting di hardware dengan mengetik sensors, tapi kalo pengen nampilin di gnome shell, masih membutuhkan beberapa hal lain..

2. install gnome-tweak, kalo belum ada
apt-get install gnome-tweak-tool

3. donlot sensors gnome shell extension dari github pake git
git https://github.com/xtranophilist/gnome-shell-extension-sensors.git

4. pindah ke folder gnome-shell-extension-sensors di terminal dan jalanin perintah ./autogen.sh

5. terus bangun dan install sensors nya pake make dan make install (butuh jadi root), kalo punya checkinstall bisa juga pake checkinstall pas installnya. terus log out dan login lagi.

6. kalo udah, tinggal nambahin ke gnome shell, menggunakan gnome tweak tool caranya buka  advance setting>>shell extension>> aktifin sensors nya, maka akan muncul di panel di atas... (btw emang panel gnome-shell kan cuma atu ...:D) dan kira2 begini hasilnya...
