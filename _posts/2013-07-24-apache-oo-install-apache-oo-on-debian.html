Apache OO Logo

OpenOffice akhirnya rilis juga versi 4 nya, setelah dilepas ke Apache Foundation oleh Oracle... kesan pertama makin mirip MS Office 2007 deh...

Nah, yang pengen nyobain install di Debian mesti siap-siap bermasalah dengan paket desktop-integrationnya, karena tabrakan sama desktop integrationnya Libreoffice. Mau uninstall desktop-integrationnya berarti uninstall semua Libreofficenya karena paketnya masuk di core paket libreoffice.. Ga install desktop-integration bawaan Apache OO, menunya ga muncul di list aplikasi... dah rempong jadinya....

 
Karena pengen nyobain, ya udah lah, uninstall dulu librenya....terus coba install lagi desktop integrationnya... dan muncul juga akhirnya di list aplikasi.. cobain, dan buka dokumen dari MS Words alias docx.. dan memang lebih halus sih... sesuai dengan reviu dari zdnet yang bilang kalo kompatibilitas dengan format ooXML nya MS Office ini lebih baik di Apache OO ver 4 ini...


Installnya gimana sih?? gampang banget kok, tinggal donlot sesuaikan dengan kebutuhan anda, kalo make Debian dan turunannya, donlot yang .deb, terus ekstrak, dan masuk ke folder .DEBS di hasil ekstrakkan tadi.. nah tinggal buka terminal disitu terus install pake dpkg -i. udah itu masuk ke folder desktop-integratio dan install juga desktop integrationnya...

thats it.... sukses, silahkan coba mainkan......
