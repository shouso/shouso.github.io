Seru juga kalo punya website sesibuk pkspiyungan.org yang sehari pageviews nya dapat mencapai 350 ribuan. Coba kalo di uangin dengan masang adsense atau pasang iklan dari amazon, coba berapa perharinya yang kite dapet dolarnya :D .... ngimpi kali ye.....

Nah, saking sibuknya, orang-orang pada melototin tuh setiap jengkal webnya om piyu, sampe dari postingannya, foto narsisnya sampe counter pageviewsnya yang ga pernah berenti tuh. Sampe-sampe sambungan internet di putusin juga entu counter tetep ngitung aja... Dasar om piyu curang... gimana ga curang coba, 350.000 pageviews kagak ada yang di kasihin ke web saya barang 10.000 perhari, kan tuh counter pageviews ga lama-lama amat ngitungnya ketika udah di putus internetnya... :D


Nah, masalah counter pageviews ini yang dijadiin bahan buat ngebully om piyu, bahwasanya om piyu itu ngakalin counternya agar terlihat yang nonton entu web jutaan. Argumennya kenapa kok internet di putus itu itungan tetep jalan aja... kira-kira begitu. Sebenernye sih udah dibahas beberapa kali sama om piyu disini tetapi om piyu ngomong kalo kagak ngarti cara kerja pageviews bawaan blogger entu seperti postingannya :

"Admin sendiri tidak punya ilmu tentang cara kerja pageviews. Kalau ditanya 'kok pageviews tetap berjalan walau koneksi internet mati?' terus terang saya gak bisa jawab (dan tulisan diatas sdr Agung Sagita Purnama mungkin bisa sedikit menjelaskan). Dan kalau ada pembaca yang bisa memberi penjelasan lain, memperkaya ilmu itu sangat ditunggu"

Sebenernya sih itu simpel aja kok, ini ada hubungannya sama web cache alias halaman web yang disimpen sementara di harddisk pc/laptop oleh web browser ente, jadi ketika ente ngebuka halaman web yang sama, entu browser ga perlu lama-lama nampilinnya, tinggal liat yang ada di harddisk ente, terus dibandingin ama yang di server web nya, kalo sama ngapain minta lagi, kan lama.

Contoh lain web cache adalah ketika nonton video di youtube.com, sebenernya ketika milih pideo di sana, itu web browser udah mulai nge donlot videonya dan ditaro di harddisk sebagai simpenan sementara, makanya kalo abis nonton youtube, kalo ente ngarti browser ente naro file pideo nya tinggal ente pindahin ke tempat lain kalo mau nonton lagi tanpa ngebuka youtube.

Masih kagak percaya, coba kita analisa gimana counter pageviewsnya webnya om Piyu. Bahan-bahan yang musti disiapin, GNU/Linux Zorin 8 amd64, browser firefox, addons firefox cache viewer ( kalo butuh), dan data yang kita punya pageviews perharinya kita ambil rata-rata 300.000, jadi per detik = 300.000/(24x60x60) dapetnya 3,4 per detiknya. Ada yang mau ngitung pake stop watch??

Nah, caranya buat ngecek berapa counter pageviews yang udah di simpen di harddisk ente, lakuin hal berikut ini :
1. paling bagus itu install firefoxnya baru, bener-bener baru, yang terbaru kalo perlu, jadi masih belum ada cache yang kesimpen di harddisk. Kalo udah punya, hapus semua cache nya dengan cara matiin firefoxnya, terus buka lagi (jangan ngebuka web dulu), klik History>>Clear Recent History terus pilih everything. Klik Clear Now.Tunggu sampe selesai.
2. Jika udah, matiin firefoxnya, terus buka lagi, dan ketik about:cache kalo masih baru mestinya masih 0 bytes ukuran cache nya seperti ini.

Klo belum 0 byte ulangi lagi langkah 1.
3. Jika sudah, buka tab baru terus ketik alamat webnya pkspiyungan.org, tunggu sampe ke load semua, sampe pageviews counternya keliatan. Jika udah, coba di refresh yang tab about:cache, mestinya udah berubah seperti gambar ini.
