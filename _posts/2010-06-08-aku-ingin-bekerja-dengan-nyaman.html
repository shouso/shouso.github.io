Aku Ingin Bekerja dengan Nyaman Menggunakan GNU/Linux


Yah, itulah keinginanku saat ini, bekerja dengan baik dan berdaya guna menggunakan Software berlisensi FOSS dan tidak perlu lagi memakai format-format MS Windows dan MS Office plus serbuan Worm, trojan, virus dan sebagainya.


Memang, tidak semua harus FOSS, tetapi setidaknya para penyedia aplikasi-aplikasi untuk bekerja semisal Metastorm Provision mau menyediakan versi Linuxnya, agar saya dapat bekerja dengan nyaman di lingkungan GNU/Linux. Bukan saya membenci MS Windows dan segala aplikasinya, tetapi saya ingin kebebasan saya kembali. Menggunakan MS Windows sebenarnya anda menyerahkan kebebasan anda untuk berekspresi kepada Microsoft Inc. Seperti contoh dari Windows Vista Ultimate EULA berikut ini :


"SCOPE OF LICENSE.    The software is licensed, not sold.    This agreement only gives you some rights to use the software.    Microsoft  reserves all other rights.    Unless applicable law gives you more rights despite this limitation, you may use the software only as expressly permitted in this agreement.    In doing so, you must comply with any technical limitations in the software that only allow you to use it in certain ways.    For more information, see http://www.microsoft.com/licensing/userights. You may not :


·  work around any technical limitations in the software;
·  reverse engineer, decompile or disassemble the software, except and only to the extent that applicable law expressly permits, despite this   limitation;
·  use components of the software to run applications not running on the software;


·  make more copies of the software than specified in this agreement or allowed by applicable law, despite this limitation;


·  publish the software for others to copy;
·  rent, lease or lend the software; or
·  use the software for commercial software hosting services. " (1)


Begitulah, sebenarnya kita hanya dipinjami oleh Microsoft Inc, tidak lebih!!! Bahkan jika ada masalah karena kekurangan softwarenya, kita tidak boleh memperbaikinya, atau untuk menggunakan bagian dari softwarenya saja tidak boleh.


Menggunakan software berlisensi FOSS, sebenarnya  mengembalikan kedaulatan anda kepada pribadi anda. Anda berhak untuk mengubah dan menyesuaikan software yang anda dapat, memperbanyaknya dan menjualnya, maupun menggunakannya untuk mengembangkan software anda sendiri. Memang sih tergantung lisensi FOSS yang digunakan, akan tetapi secara umum anda memiliki copy dari software yang anda dapatkan, sama sekali berbeda dengan EULA nya Microsoft yang berkata softwarenya hanya licensed not sold (2).


Memang, aplikasi di GNU/Linux dan FOSS secara umum ada yang belum dapat menggantikan software proprietary, tetapi setidaknya dengan semakin banyak pemakai GNU/Linux, pengembang akan berfikir ulang untuk mulai membuat versi GNU/Linux nya. Dan tentunya akan sangat menyenangkan jika di kantor, di rumah, di sekolah, makin banyak orang yang menggunakan linux.


Sebenarnya, banyak aplikasi generik GNU/Linux yang dapat menggantikan aplikasi proprietay berbasis MS Windows semisal MS Office dapat digantikan oleh OpenOffice, Adobe Photoshop dapat digantikan oleh GIMP, MS Windows XP dapat digantikan oleh Debian GNU/Linux atau Ubuntu, dll (3), tinggal penggunanya saja yang menyesuaikan dngan perubahan dan transisi dari MS Windos ke GNU/Linux.
